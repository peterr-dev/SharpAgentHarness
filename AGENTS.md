# Guidance for coding agents

This repo is a minimal, general-purpose, tool-using agent harness. It was created to learn how agent harnesses work, and provide a foundation for experimenting with agentic concepts.

## Languages
- Use vanilla JavaScript, HTML and CSS for web UI.

## Code Style
- Write code for education, clarity and ease of understanding - code should be easily understood by other developers. 
- Don’t use fancy techniques which obscure the core concepts.
- Write comments for key sections of code, and always use British English.
- Declare constants if it makes the code more understandable e.g. magic values.

## UX
- Style the UI inspired by OpenAI's style (light theme).
- Wrap all API calls from the web UI, to catch errors and visualise them for the user.
